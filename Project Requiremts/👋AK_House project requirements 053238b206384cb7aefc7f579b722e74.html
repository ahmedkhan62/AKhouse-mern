Registration
  • Registration should be done using the OTP system. There can be sent using the phone number or email address.
  • Need to collect following info when registering:
    • Full name
    • Profile picture
    • Create username

Rooms
  • Authentication user can see list of available public rooms
  • User can filter the rooms using search box
  • User can join existing room or create a new room
  • Creator of the room can add other users as speakers
  • Room should have following types:
    • Public room:

      Any Authenticated user can see this room and can join that room

    • Closed room:

      This room can be joined only through the link.

Room:

Room will have 2 types of users

  1. Speakers

    People who can speak. ( they can mute/unmute themselves )

  1. Listeners

    People who can only listen. ( their mic will be muted by default )

  • Any user can leave the room.

    Speaker can choose their Mic from available devices.

    Only creator of the room can delete the room.

Profile
  • User will have a user profile page. Any authenticated user can visit this page and follow that user.
  • User should have followers and following count on the profile page.

Login
  • User can login using OTP.
  • OTP will be sent on phone number or email address.

Logout
  • User can logout using logout button.
